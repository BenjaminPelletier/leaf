What is Fony?
-------------
Fony is an editor for Windows bitmap fonts. If you don't know what bitmap
fonts are, chances are you won't need this program.


How much does it cost?
----------------------
Nothing.


Will you add support for filetype X or feature Y?
-------------------------------------------------
It won't hurt to ask. A small donation wouldn't hurt either.
